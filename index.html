jQuery Notify plugin

Wait for 3 notifications to appear on your screen!

This plugin uses HTML storage to save showed messages


This demo page uses this settings: $(function () { $().notify({ data: [ { // This message will appear 1 sec. after document is loaded "name": "Hello World 1", "body": " This is only example message ", "icon": "notify.png", "time": (Math.floor(Date.now() / 1000) + 1) }, { // This message will appear 5 sec after document is loaded "name": "Hello World 2", "body": " This is only example message ", "icon": "notify.png", "time": (Math.floor(Date.now() / 1000) + 5) }, { // This message will appear 10 sec after document is loaded "name": "Hello World 3", "body": " This is only example message ", "icon": "notify.png", "time": (Math.floor(Date.now() / 1000) + 10) } ] }); });
